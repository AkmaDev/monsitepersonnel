# Manassé A. AKPOVI

Je construis des produits numériques pour les gens que le numérique oublie.

---

### BibleFon — *[biblefon.org](https://biblefon.org)*

Histoires bibliques illustrées, racontées en langue fon.
Une grand-mère béninoise peut écouter David et Goliath dans sa langue maternelle
sans avoir besoin de lire — sans connexion internet, depuis son téléphone.

C'est ça, le numérique inclusif.

**Stack :** Next.js · TypeScript · TailwindCSS · Meta MMS-TTS-Fon · HuggingFace · PWA offline-first

---

### Ce que je construis

- Produits à destination des communautés **sous-représentées dans le numérique**
- Interfaces **Audio First** pour les contextes à tradition orale
- MVPs qui répondent à de vraies contraintes — pas des demos de conférence
